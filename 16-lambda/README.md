# 16 Lambda表达式

## 学习目标
掌握Lambda表达式的语法和应用场景。

## 主要内容
- Lambda表达式的语法结构
- Lambda与普通函数的区别
- Lambda在高阶函数中的应用
- 配合map()、filter()、reduce()使用
- Lambda在排序中的应用
- Lambda的限制和注意事项
- 何时使用Lambda，何时使用普通函数

## 练习要点
1. 掌握Lambda表达式的基本语法
2. 学会在map、filter等函数中使用Lambda
3. 练习Lambda在数据处理中的应用
4. 理解Lambda的适用场景和限制
5. 学会选择Lambda还是普通函数
